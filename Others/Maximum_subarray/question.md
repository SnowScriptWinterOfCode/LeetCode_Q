# Maximum Subarray(53)

Given an integer array nums, find the 
subarray with the largest sum, and return its sum.

 

## Example 1:

Input: nums = [-2,1,-3,4,-1,2,1,-5,4]

Output: 6

#### Explanation: 
The subarray [4,-1,2,1] has the largest sum 6.

## Constraints:

1 <= nums.length <= 105

-104 <= nums[i] <= 104
